## スコアの色分け

- **🌟 80点以上（緑色）**: 優秀なビルド
- **⭐ 60-80点（青色）**: 良好なビルド
- **✨ 40-60点（黄色）**: 普通のビルド
- **💫 40点未満（橙色）**: 改善が必要なビルド

## リアルタイム更新

キャラクター選択、素質選択、ロスレコ選択を変更すると、
評価スコアが即座に再計算されて表示が更新されます。

## 計算方法（現在の実装）

簡易的な評価アルゴリズムを実装：
- 素質の合計レベル
- ロスレコの装備数
- これらを組み合わせて各スコアを算出

将来的には、実際のキャラクターステータスや
ダメージ計算エンジンと統合してより精密な評価を行う予定です。
